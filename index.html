O que é Git, o que é GitHub e qual a diferença

O que é Git

Git é um sistema de controle de versão distribuído utilizado para rastrear alterações em arquivos.2026.42

O que é GitHub

GitHub é uma plataforma online que hospeda repositórios Git e facilita colaboração.

Diferença

Git é uma ferramenta local, GitHub é um serviço online.
